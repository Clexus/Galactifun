# Galactifun-星际

Slimefun-Addon-Galactifun里面加入了星系,恒星系统,行星,卫星,火箭和宇航服.
它有一个强大的开发API.
灵感来自 Claytech(粘土科技) 和 GalactiCraft(星际) Mod.
